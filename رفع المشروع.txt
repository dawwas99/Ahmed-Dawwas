خطوات رفع مشروع بايثون على GitHub باستخدام Git:

1. **إعداد Git على جهازك:**
   إذا لم تكن قد قمت بتكوين Git على جهازك بعد، يمكنك تنزيله وتثبيته من الموقع الرسمي: https://git-scm.com/

2. **إنشاء مستودع (Repository) على GitHub:**
   - قم بتسجيل الدخول إلى حسابك على GitHub.
   - انقر على زر "New" (جديد) في الزاوية العلوية اليمنى للشاشة.
   - اختر اسمًا للمستودع واختيارات أخرى مثل الوصف ونوع المستودع (عام أو خاص).
   - انقر على "Create repository" (إنشاء المستودع).

3. **التحضير للرفع:**
   - قم بفتح مجلد مشروع البايثون على جهازك باستخدام موجه الأوامر (Command Line) أو ترمينال.

4. **الربط بمستودع GitHub:**
   - استخدم الأمر التالي لربط مشروعك المحلي مع المستودع الذي أنشأته على GitHub:
   ```
   git init
   git add .
   git commit -m "أول رفع لمشروع بايثون"
   git branch -M main
   git remote add origin رابط-مستودع-GitHub.git
   ```

5. **الرفع إلى GitHub:**
   - استخدم الأمر التالي لرفع مشروعك إلى GitHub:
   ```
   git push -u origin main
   ```

6. **تحديثات المشروع:**
   - عندما تقوم بإجراء تعديلات على مشروعك، قم بتنفيذ الخطوات التالية:
   ```
   git add .
   git commit -m "وصف التعديلات"
   git push
   ```

هذه هي الخطوات الأساسية لرفع مشروع بايثون إلى GitHub باستخدام Git. تذكر أنه يمكنك تخصيص هذه الخطوات وفقًا لاحتياجات مشروعك.
